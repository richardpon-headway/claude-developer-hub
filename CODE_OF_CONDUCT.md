# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant], a
widely-used code of conduct for open-source communities.

The full text is published at the link above. By participating in this
project — opening issues, submitting pull requests, engaging in
discussions — you agree to abide by it.

## Reporting

To report a violation, open a GitHub issue (or, for sensitive reports,
follow the disclosure process in [SECURITY.md](SECURITY.md), which has
the same private-contact channel).

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
